# 插入排序

```java
public static void insertionSort(int[] arr) {
    final long start = System.currentTimeMillis();
    // i 从 1 开始，因为对第一个元素来说它已经是有序的了
    for (int i = 1; i < arr.length; i++) {
        // 只需要比较当前元素与之前排好序的值 进行比对
        for (int j = i; j > 0 && arr[j] < arr[j - 1]; j--) {
            // 判断当前元素 与 前一个元素比较
            int temp = arr[j - 1];
            arr[j - 1] = arr[j];
            arr[j] = temp;
        }
    }
    final long end = System.currentTimeMillis();
    System.out.println("插入排序耗时：\t" + (end - start));
    System.out.println(Arrays.toString(arr));
}
```

## 优化插入排序

优化插入排序：

1. 让内层循环 减少交换次数
2. 提前结束
3. 当 `arr` 近乎连续的时候，其效率高与 $NlogN$ 的算法

```java
public static void insertionSortOptimize(int[] arr) {
    for (int i = 1; i < arr.length; i++) {
        int j = i;
        int e = arr[i];
        // 插入排序可以提前终止内层循环
        for (;  j > 0 && arr[j - 1] > e; j--) {
            arr[j] = arr[j - 1];
        }
        arr[j] = e;
    }
    System.out.println(Arrays.toString(arr));
}
```

# 选择排序

```java
public static void selectionSort(int[] arr) {
    final long start = System.currentTimeMillis();
    for (int i = 0; i < arr.length; i++) {
        // 找到 [i,n) 最小值
        int minIndex = i;
        for (int j = i + 1; j < arr.length; j++) {
            if(arr[j] < arr[minIndex]) {
                minIndex = j;
            }
        }
        int temp = arr[minIndex];
        arr[minIndex] = arr[i];
        arr[i] = temp;
    }
    final long end = System.currentTimeMillis();
    System.out.println("选择排序耗时：\t" + (end - start));
    System.out.println(Arrays.toString(arr));
}
```
